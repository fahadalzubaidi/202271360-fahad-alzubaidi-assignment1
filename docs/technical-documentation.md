# Technical Documentation

## Tech Stack

- **HTML5** for semantic structure
- **CSS3** for layout, theming, and responsive design
- **JavaScript** for theme toggling and basic form interaction

## Key Features

- Dark-first design using CSS custom properties with a light-mode override.
- Responsive grids for project cards and contact section.
- Theme toggle stored in localStorage to preserve user preference.
- Smooth scrolling navigation with active link highlighting.
- Client-side contact form validation (no backend).
- Modern animations and transitions for enhanced user experience.

## File Overview

- **index.html**: Main page with About, Projects, Skills, and Contact sections.
- **css/styles.css**: Visual system (colors, typography, layout, responsive rules).
- **js/script.js**: Theme toggle, smooth scrolling, form interaction, and active link detection.
- **docs/ai-usage-report.md**: Documentation of AI assistance.
- **assets/images/**: Project images and profile photo.

## Customization

- Replace the profile placeholder in the About section with a real image in `assets/images/`.
- Update project descriptions or add new cards in the Projects section.
- Adjust colors by changing the CSS variables at the top of `css/styles.css`.
- Modify skills and their progress levels in the Skills section.
- Update contact information (email, phone, location) in the Contact section.
